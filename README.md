## 📋 Question Overview

### 🔹 Question1 - Palindrome Checker  
A C# program to check whether a given string is a palindrome (same forward and backward).

---

### 🔹 Question2 - Largest Number in Array  
A program to find the largest number from an array using basic comparison logic.

---

### 🔹 Question3 - Employee Encapsulation  
Created an `Employee` class with properties:  
- Id  
- Name  
- Salary  

Demonstrates encapsulation using properties with getters and setters.

---

### 🔹 Question4 - LINQ Filtering  
Used a LINQ query to filter employees whose salary is greater than 50,000.

---

### 🔹 Question5 - Simple API  
Built a simple ASP.NET Core Web API with:  
- GET method → Retrieve data  
- POST method → Add data  

---

### 🔹 Question6 - SQL (Duplicate Records)  
Wrote a SQL query using `GROUP BY` and `HAVING` to find duplicate records in a table.

---

### 🔹 Question7 - Dependency Injection (SampleDI)  
Implemented Dependency Injection in ASP.NET Core using:  
- Interface (`IEmployeeService`)  
- Service class (`EmployeeService`)  
- Registered in `Program.cs`
